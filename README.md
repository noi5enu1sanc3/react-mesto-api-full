## Run Locally

Clone the project

```bash
  git clone git@github.com:noi5enu1sanc3/react-mesto-api-full.git
```

Go to the project directory

```bash
  cd react-mesto-api-full
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
